# Jelly Novel - Premium Story Platform

A Next.js 14 application focusing on immersive reading experiences using Framer Motion and Physics-based interactions.

## Features
- **Hero Section**: 3D Floating Book, Aurora Backgrounds.
- **Story Grid**: Bento-style layout, Glassmorphism, Micro-interactions.
- **Reading View**: Seamless transitions (Shared Layout), Sticky Progress, Typography optimized for reading.
- **Tech Stack**: Next.js 14, Tailwind CSS, Framer Motion, Lucide React.

## Getting Started

1. Install dependencies:
   ```bash
   npm install framer-motion lucide-react clsx tailwind-merge
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000).
